Samyati is a travel blogging website. 
Tech Stack used: MERN Stack

Purpose: Built for a MERN Stack certification course

- The .env file is excluded in the github repository, so those whoever wishes to clone the repo has to define their own .env file for both client and server.
- Entire database was stored locally... so new system requires new datasets.
- The Website has 4 main pages excluding signin and signup page.
- Signin and signup are done using clerk authentication API, to help integrate google and other means to login to a webpage.
- User roles are divided into author and users. Where a person who write a particular blog is author to that blog and is the only person who can edit it.
